# GitHub OAuth App with Express & Passport.js

A simple web application demonstrating GitHub OAuth authentication using Express.js and Passport.js.

## Features

- 🔐 GitHub OAuth 2.0 authentication
- 🎯 Session management with express-session
- 📊 User profile display after login
- 🎨 EJS templating for views
- 🚀 Easy setup and deployment

## Prerequisites

Before running this application, make sure you have:

- Node.js (v14 or higher)
- A GitHub account
- A registered GitHub OAuth Application
